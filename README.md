# Horizon-Alpha

🚀 Predictive Business Simulation Engine

Horizon-Alpha is a Monte Carlo simulation engine that predicts financial trajectories based on starting capital, revenue, expenses, and growth rates.

## Features
- Monte Carlo simulation
- Interactive Plotly chart
- KPI cards
- Risk analysis
- Dark-mode UI

## Running the App
1. Create a virtual environment and activate it
2. Run `pip install -r requirements.txt`
3. Run `streamlit run app.py`
